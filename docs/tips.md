# Tips
> Install and configure


## APT

```sh
$ apt search TERM
```

Check dependencies

```sh
$ apt cache depends PACKAGE		
```

The packages list can be viewed through a GUI. Or through the file directly:

```sh
$ view /etc/apt/sources.list
```
